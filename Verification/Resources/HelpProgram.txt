{\pard\sa180\fs24 Работа с программой\par}
{\pard\b Добавление модели \b0\par}
{\pard Для добавления новой модели нажмите на кнопку "Добавить" и выберите необходимые файлы. Одновременно возможно добавлять любое количество моделей. \par}
{\pard Для установления связи между XMI файлом и PNG изображением необходимо, чтобы имена файлов были одинаковыми. \par}
{\pard \par}
{\pard\b Удаление модели \b0\par}
{\pard Для удаления модели выберите удаляемую модели из списка диаграмм, нажмите кнопку "Удалить". \par}
{\pard \par}
{\pard\b Верификация моделей \b0\par}
{\pard Для верификации загруженных моделей нажмите кнопку "Верифицировать". Процесс верификации затрагивает все загруженные и еще не проверенные модели. Результаты верификации выводятся в таблице ошибок, а также записываются в лог в \b"\\results\\verificationResults.txt"\b0 в папке с программой. \par}
{\pard Если XMI файл модели содержит координаты элементов и для модели было загруженно изображение, то для некоторых ошибок на изображении выделяются связанные с ними элементы. \par}
{\pard \par}
{\pard\b Сохранение ошибок в файл \b0\par}
{\pard Для сохранения списка ошибок в текстовый файл нажмите кнопку "Экспорт ошибок" и выберите месторасположение файла для сохранения. \par}
